× RGBmatic Instructions

The object of the game is to match the rgb numeric color value listed in the top header with one of the colors shown below.

To play, simply select the colored square that you think is the match to the rgb value.

Tip: Remember the numeric value shown gives you the amount of red, green, and blue (rgb) used to create the color. On a scale of 0 to 256, the higher the numeric value, the more dominant the color will be.
